Installation Steps

1

Download Extension

Download or clone the extension files to your computer. The extension is currently in development mode.

2

Open Chrome Extensions

Navigate to chrome://extensions/ in Chrome and enable "Developer mode" in the top right corner.

3

Load Extension

Click "Load unpacked" and select the extension directory. The Prompt Library icon will appear in your toolbar.

4

Start Using

Visit any LLM platform or press Alt+P anywhere to open the panel and start building your prompt library.

How It Works

1

Panel Injection

When you click the extension icon or press Alt+P, a right-side panel slides in using Shadow DOM for style isolation.

2

Smart Detection

The extension uses multiple selector strategies to find text input areas on each LLM platform, with robust fallbacks.

3

Seamless Insertion

Prompts are inserted directly into the composer with proper DOM events. If insertion fails, content is copied to clipboard.

4

Go-To LLM

When no LLM is open, the extension can automatically open your preferred platform and insert the prompt.
